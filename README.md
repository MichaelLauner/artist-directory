# Artist Directory

Public artist directory templates and filtering powered by Directory Core.

Publisher: Mostly Wanted

Status: Beta (`0.1.10-beta.1`). This plugin is public for collaboration and client testing, but the display options, templates, and submission/approval roadmap are still evolving.

## Current Scope

- public artist archive
- media filters
- public artist profile pages
- theme template overrides via `artist-directory/`

## Dependency

This plugin requires `Directory Core` to provide the underlying artist and venue records.
